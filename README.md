SportKing Intro Animation
A dynamic full-screen web intro for SportKing featuring a 3D cricket ball smash effect with glass shards, flash impact, and animated logo reveal—perfect for gaming/sports sites.

Features
Realistic cricket ball throw with spin, blur, and scale using CSS 3D transforms.

Screen shatter effect: 50 randomized glass shards scatter with staggered timing.

Flash impact overlay and golden-highlighted "SportKing" logo expansion.

Auto-redirect to landing_page.php after 4.5s (with fallback confirm).

Tech Stack
HTML5/CSS3: Animations via @keyframes, perspective, clip-path for shards.
Enable Arduino IDE with ESP8266, create a mini server, and access it without internet.

Vanilla JavaScript: Dynamic shard creation, positioning, and timing.

Getting Started
Clone/download the repo.

Open index.html in any modern browser (Chrome/Firefox recommended).

Ensure landing_page.php exists in the same folder for redirect, or modify JS timeout.



Usage
Serve via any static host (GitHub Pages/Netlify) or PHP server for the redirect. Mobile-responsive with viewport sizing.
